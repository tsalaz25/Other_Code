Readme For Project6.zip
Usage: Run the Script 'funtionApprox.m' with the function files 'NCS.m', 'Eval.m', and 'Coef.m'

Description of Files

-Eval.m and Coef.m 
Given in-class, have been used in previous projects
 
-NCS.m 
Function file that defines the process for creating a Natural Cubic Spline. Inputs and Outputs are describes in the file, Follows the process we used in class to create a cubic spline, Comments outline the process

-functionApprox.m
Script file for approximating tanh(x)
IMPORTANT: lines 7-9 define the Interval and the number of Nodes. lines 17-20 define the Interval for Smooth Plotting of the function. 
linspace(x,y,z) is used in, be sure that x,y arguments are the same in both. In x_nodes, the z defines the amount of nodes and in x_vals z is used for plotting.
